# Tech4Humans - Backend

Este projeto foi desenvolvido com [NodeJs](https://nodejs.org/en/).

## Comandos

No diretório do projeto, execute:

### `npm install`

Para baixar as dependencias

```
npm install
```

### `npm start`

Para iniciar o sistema

```
npm start
```
